## Usage

There is no UI.  To input an unsolved sudoku, edit the array `sudoku` near the top of the program.  Zeroes are blank spaces.

## Compilation

No special compile parameters

```sh
sudo apt install build-essential
gcc sudoku.c
./a.out
```
